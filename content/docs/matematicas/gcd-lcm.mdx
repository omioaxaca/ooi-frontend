## M√°ximo Com√∫n Divisor (MCD/GCD)

El **M√°ximo Com√∫n Divisor** (MCD o GCD en ingl√©s) de dos n√∫meros es el n√∫mero m√°s grande que divide a ambos.

### Ejemplos

- $\gcd(12, 18) = 6$ porque los divisores de 12 son {1, 2, 3, 4, 6, 12} y los de 18 son {1, 2, 3, 6, 9, 18}. El mayor com√∫n es 6.
- $\gcd(15, 25) = 5$
- $\gcd(7, 13) = 1$ (son coprimos)

### M√©todo ingenuo

Verificar todos los divisores posibles:

```cpp
int gcdIngenuo(int a, int b) {
    int resultado = 1;
    for (int i = 1; i <= min(a, b); i++) {
        if (a % i == 0 && b % i == 0) {
            resultado = i;
        }
    }
    return resultado;
}
```

**Complejidad:** $O(\min(a, b))$ - Muy lento para n√∫meros grandes.

## Algoritmo de Euclides

El **Algoritmo de Euclides** es uno de los algoritmos m√°s antiguos conocidos (¬°m√°s de 2300 a√±os!) y es extremadamente eficiente.

### La idea clave

Se basa en esta propiedad fundamental:

$$\gcd(a, b) = \gcd(b, a \mod b)$$

¬øPor qu√© funciona? Si $d$ divide tanto a $a$ como a $b$, entonces $d$ tambi√©n divide a $a - kb$ para cualquier $k$. En particular, divide a $a \mod b = a - \lfloor a/b \rfloor \cdot b$.

### Ejemplo paso a paso

Calcular $\gcd(48, 18)$:

```
gcd(48, 18):  48 mod 18 = 12  ‚Üí  gcd(18, 12)
gcd(18, 12):  18 mod 12 = 6   ‚Üí  gcd(12, 6)
gcd(12, 6):   12 mod 6 = 0    ‚Üí  gcd(6, 0)
gcd(6, 0):    resultado = 6
```

### Implementaci√≥n recursiva

```cpp
int gcd(int a, int b) {
    if (b == 0) return a;
    return gcd(b, a % b);
}
```

### Implementaci√≥n iterativa

```cpp
int gcd(int a, int b) {
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}
```

### Usando la STL (C++17+)

```cpp
#include <numeric>  // Para std::gcd

int main() {
    int a = 48, b = 18;
    cout << __gcd(a, b) << endl;  // Funciona en todas las versiones
    cout << gcd(a, b) << endl;     // C++17+
    return 0;
}
```

**Complejidad:** $O(\log(\min(a, b)))$ - ¬°Muy r√°pido!

<details>
<summary>üí° ¬øPor qu√© es tan eficiente?</summary>

En cada paso, al menos uno de los n√∫meros se reduce a la mitad. Si $a \geq b$, entonces $a \mod b < a/2$. Esto significa que el algoritmo hace a lo m√°s $2 \log_2(\min(a, b))$ operaciones.

</details>

## M√≠nimo Com√∫n M√∫ltiplo (MCM/LCM)

El **M√≠nimo Com√∫n M√∫ltiplo** (MCM o LCM en ingl√©s) de dos n√∫meros es el n√∫mero m√°s peque√±o que es divisible por ambos.

### Relaci√≥n con el MCD

Existe una relaci√≥n fundamental:

$$\text{lcm}(a, b) = \frac{a \cdot b}{\gcd(a, b)}$$

### Implementaci√≥n

```cpp
long long lcm(long long a, long long b) {
    return a / __gcd(a, b) * b;  // Divisi√≥n primero para evitar overflow
}
```

> ‚ö†Ô∏è **Importante:** Hacemos `a / gcd(a, b) * b` en lugar de `a * b / gcd(a, b)` para evitar overflow cuando $a \cdot b$ es muy grande.

### Ejemplo

$\text{lcm}(12, 18) = \frac{12 \cdot 18}{6} = \frac{216}{6} = 36$

Verificaci√≥n: 36 es divisible por 12 y por 18, y no hay n√∫mero menor con esa propiedad.

## Algoritmo de Euclides Extendido

El algoritmo extendido no solo calcula $\gcd(a, b)$, sino que tambi√©n encuentra enteros $x$ e $y$ tales que:

$$ax + by = \gcd(a, b)$$

Esto se conoce como la **Identidad de B√©zout**.

### Implementaci√≥n

```cpp
int gcdExtendido(int a, int b, int &x, int &y) {
    if (b == 0) {
        x = 1;
        y = 0;
        return a;
    }

    int x1, y1;
    int g = gcdExtendido(b, a % b, x1, y1);

    x = y1;
    y = x1 - (a / b) * y1;

    return g;
}
```

### Ejemplo

Para $\gcd(48, 18) = 6$, el algoritmo encuentra $x = -1$ e $y = 3$:

$$48 \cdot (-1) + 18 \cdot 3 = -48 + 54 = 6$$

### Aplicaciones

1. **Encontrar inversos modulares** (lo veremos en aritm√©tica modular)
2. **Resolver ecuaciones diof√°nticas lineales**
3. **Criptograf√≠a** (RSA)

## Propiedades importantes

### Propiedades del MCD

1. $\gcd(a, 0) = a$
2. $\gcd(a, a) = a$
3. $\gcd(a, b) = \gcd(b, a)$ (conmutativa)
4. $\gcd(a, b, c) = \gcd(\gcd(a, b), c)$ (asociativa)
5. $\gcd(ka, kb) = k \cdot \gcd(a, b)$
6. Si $\gcd(a, b) = 1$, decimos que $a$ y $b$ son **coprimos**

### Propiedades del MCM

1. $\text{lcm}(a, 1) = a$
2. $\text{lcm}(a, a) = a$
3. $\text{lcm}(a, b) = \text{lcm}(b, a)$
4. $\text{lcm}(a, b, c) = \text{lcm}(\text{lcm}(a, b), c)$
5. $\gcd(a, b) \cdot \text{lcm}(a, b) = a \cdot b$

## GCD/LCM de m√∫ltiples n√∫meros

### MCD de un arreglo

```cpp
int gcdArreglo(vector<int>& arr) {
    int resultado = arr[0];
    for (int i = 1; i < arr.size(); i++) {
        resultado = __gcd(resultado, arr[i]);
        if (resultado == 1) break;  // No puede ser menor
    }
    return resultado;
}
```

### MCM de un arreglo

```cpp
long long lcmArreglo(vector<int>& arr) {
    long long resultado = arr[0];
    for (int i = 1; i < arr.size(); i++) {
        resultado = resultado / __gcd(resultado, (long long)arr[i]) * arr[i];
    }
    return resultado;
}
```

## Problemas de pr√°ctica

### Problema 1: GCD de consultas

Dado un arreglo, responde consultas del GCD de rangos.

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, q;
    cin >> n >> q;

    vector<int> arr(n);
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }

    // Tabla dispersa para consultas de GCD
    int LOG = log2(n) + 1;
    vector<vector<int>> tabla(LOG, vector<int>(n));

    // Caso base
    for (int i = 0; i < n; i++) {
        tabla[0][i] = arr[i];
    }

    // Llenar tabla
    for (int j = 1; j < LOG; j++) {
        for (int i = 0; i + (1 << j) <= n; i++) {
            tabla[j][i] = __gcd(tabla[j-1][i], tabla[j-1][i + (1 << (j-1))]);
        }
    }

    // Responder consultas
    while (q--) {
        int l, r;
        cin >> l >> r;
        l--; r--;  // 0-indexed

        int k = log2(r - l + 1);
        int resultado = __gcd(tabla[k][l], tabla[k][r - (1 << k) + 1]);
        cout << resultado << "\n";
    }

    return 0;
}
```

### Problema 3: Ecuaci√≥n Diof√°ntica

Encuentra una soluci√≥n entera a $ax + by = c$ (si existe):

```cpp

int gcdExtendido(int a, int b, int &x, int &y) {

int main() {

    cin >> a >> b >> c;

    int x, y;
    int g = gcdExtendido(abs(a), abs(b), x, y);

    if (c % g != 0) {
        cout << "No hay soluci√≥n" << endl;
    } else {
        x *= c / g;
        y *= c / g;

        if (a < 0) x = -x;
        if (b < 0) y = -y;

        cout << "x = " << x << ", y = " << y << endl;
        cout << "Verificaci√≥n: " << a << "*" << x << " + " << b << "*" << y << " = " << a*x + b*y << endl;
    }

    return 0;
}
```
